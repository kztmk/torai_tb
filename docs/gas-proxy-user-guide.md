# GAS連携の本人確認と安全なデータ同期について

## 概要

虎威では、Google Spreadsheet と連携するために Google Apps Script（GAS）のWebアプリURLを登録します。

今回の変更により、GAS連携時に本人確認コードとFirebase UIDを使った確認を行うようになりました。これにより、登録したSpreadsheetを操作できるのは、そのSpreadsheetを連携した虎威アカウントだけになります。

## 何が変わったか

以前は、虎威にGASのWebアプリURLを登録すると、そのURLを使ってデータ同期を行っていました。

現在は、初回登録時に次の確認を行います。

1. Spreadsheetの `虎威連携` メニューから本人確認コードを生成する
2. 虎威のプロフィール画面にGAS WebアプリURLと本人確認コードを入力する
3. 虎威がFirebase UIDと本人確認コードをGASへ送信する
4. GASが本人確認コードを検証し、その虎威アカウント専用の連携情報を発行する
5. 以後のデータ同期は、虎威のサーバー側で署名してからGASへ送信する

このため、ブラウザから直接GASへアクセスしたり、他のユーザーが同じURLだけを使って操作したりすることはできません。

## 利用者が行うこと

### 初回連携

1. 自分のGoogle Spreadsheetを開きます。
2. Spreadsheetを再読み込みします。
3. メニューの `虎威連携` から `本人確認コードを生成` を選びます。
4. 表示された本人確認コードをコピーします。
5. 虎威にログインします。
6. プロフィールのAPIキー設定を開きます。
7. `Google Sheets URL` にGAS WebアプリURLを入力します。
8. `GAS本人確認コード` にコピーしたコードを入力します。
9. 保存します。

保存に成功すると、以後は通常どおり投稿データ、投稿済みデータ、エラーデータ、Xアカウント情報を同期できます。

### 2回目以降

本人確認コードの再入力は通常不要です。

ただし、次の場合は再連携が必要になることがあります。

- GASを別のSpreadsheetへ作り直した
- GAS WebアプリURLを変更した
- GAS側の連携情報を初期化した

別の虎威アカウントへの連携移転はできません。どうしても別アカウントで同じ内容を使いたい場合は、元のSpreadsheetのデータだけを新しいSpreadsheetへコピーし、新しいSpreadsheetと新しいGAS Webアプリとして連携してください。

## 安全性について

今回の仕組みでは、GASとの通信に使う秘密情報はブラウザには保存されません。

秘密情報は虎威のサーバー側で管理され、通常のデータ同期時にはサーバーが毎回署名を作成します。GAS側は、その署名とFirebase UIDを確認してから処理します。

## よくあるエラー

### `GAS proxy authorization is not initialized for this user.`

GAS本人確認コードによる初期連携が完了していません。

プロフィールのAPIキー設定で、GAS WebアプリURLと本人確認コードを入力して保存してください。

### `Firebase UID is not authorized for this spreadsheet.`

このSpreadsheetは別の虎威アカウントに連携されています。

現在ログインしている虎威アカウントが正しいか確認してください。別アカウントへの連携移転はできません。別アカウントで使う必要がある場合は、元のSpreadsheetのデータだけを新しいSpreadsheetへコピーし、新しいSpreadsheetと新しいGAS Webアプリとして連携してください。

### `Invalid setup code.`

本人確認コードが正しくありません。

GAS側で新しい本人確認コードを生成し、虎威に入力してください。

### `Setup code has expired.`

本人確認コードの有効期限が切れています。

GAS側で新しい本人確認コードを生成してください。

本人確認コードはSpreadsheetメニューの `虎威連携` から再生成できます。

### SpreadsheetAppの権限エラー

GASがSpreadsheetを操作する権限をまだ承認していない可能性があります。

GAS側で必要な権限を承認し、Webアプリを再デプロイしてください。

## 注意点

- GAS WebアプリURLだけでは連携は完了しません。
- 本人確認コードは初回連携用です。他人に共有しないでください。
- `虎威連携` メニューが表示されない場合は、Spreadsheetを再読み込みしてください。
- Google Spreadsheetの所有者、GASのデプロイ設定、虎威にログインしているアカウントが想定どおりか確認してください。
- GASを更新した場合は、必要に応じてWebアプリを再デプロイしてください。
